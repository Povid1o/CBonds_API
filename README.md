# Интеграция с CBonds JSON API

Удобный Python-клиент (`CBondsAPI`) для взаимодействия с JSON API информационного агентства CBonds. Библиотека инкапсулирует особенности авторизации, сглаживает неконсистентность именования полей API и предоставляет удобные обертки для работы с облигациями, котировками и макроэкономическими показателями.

---

## 🚀 Быстрый старт

### 1. Установка зависимостей
Убедитесь, что вы находитесь в виртуальном окружении, и установите библиотеки из файла зависимостей:
```bash
source venv/bin/activate
pip install -r requierements.txt
```

### 2. Настройка учетных данных
Создайте файл `.env` в корне проекта (он автоматически добавлен в `.gitignore`) и добавьте ваши учетные данные CBonds:
```env
CBONDS_LOGIN=ваш_логин
CBONDS_PASSWORD=ваш_пароль
```

### 3. Проверка работы
Запустите тестовый скрипт, чтобы убедиться, что подключение и запросы работают корректно:
```bash
python test.py
```

## 🛠️ Как это работает изнутри

### 1. Авторизация и запросы
Все запросы выполняются методом **POST** на базовый адрес `https://ws.cbonds.info/services/json/?lang=rus`. 
При инициализации класса `CBondsAPI` автоматически вызывается функция `load_dotenv()`, считывающая переменные из `.env`. 
Каждый запрос содержит JSON-тело следующей структуры:
```json
{
  "auth": {"login": "...", "password": "..."},
  "filters": [...],
  "quantity": {"limit": 50, "offset": 0},
  "sorting": [...],
  "fields": []
}
```

### 2. Обработка ошибок
Даже в случае сбоя API CBonds может возвращать HTTP-статус `200 OK`. Наша библиотека проверяет JSON-ответ на наличие ключа `"error"` в корне структуры. Если ошибка найдена (например, `err_no` или `err_str`), клиент выбрасывает исключение `RuntimeError`.

### 3. Решение проблемы неконсистентных фильтров (ISIN)
В зависимости от эндпоинта, API CBonds требует разного именования поля для фильтрации по ISIN:
* В методах параметров облигаций (`get_emissions`) и биржевых сделок (`get_tradings_new`) используется имя поля `"isin_code"`.
* В методах денежных потоков (`get_flow_new`), оферт (`get_offert`) и дефолтов (`get_emission_default`) используется `"emission_isin_code"`.

**Класс `CBondsAPI` автоматически подставляет нужное имя поля в зависимости от вызываемого метода.**

## 📚 Возможности библиотеки и примеры использования

### Инициализация клиента
```python
from cbonds_api_test import CBondsAPI

# Логин и пароль будут автоматически считаны из .env
api = CBondsAPI()

# Альтернативный вариант (передача параметров напрямую):
# api = CBondsAPI(login="custom_login", password="custom_password")
```

---

### 1. Данные по облигациям и эмиссиям

#### Получить общую информацию об облигации (`get_bond_info`)
Возвращает подробные параметры выпуска по его ISIN-коду (использует эндпоинт `get_emissions`).
*Название бумаги хранится в ключе `document_rus`, а имя эмитента — в `emitent_name_rus`.*
```python
bond = api.get_bond_info("RU000A10DQA8")
print(f"Название: {bond.get('document_rus')}")
print(f"Дата погашения: {bond.get('maturity_date')}")
```

#### Поиск облигаций (`search_bonds`)
Поиск списка облигаций по префиксу ISIN-кода (по умолчанию `RU000A`) и валюте выпуска.
```python
# Поиск до 10 рублевых облигаций
bonds = api.search_bonds(isin_prefix="RU000A", currency="RUB", limit=10)
for b in bonds:
    print(b.get("isin_code"), b.get("document_rus"))
```

#### График выплат (купоны и амортизация) (`get_bond_cashflows`)
Возвращает будущие и прошлые купонные выплаты, а также номинальные амортизации по облигации (эндпоинт `get_flow_new`).
*Сумма купона находится в поле `cupon_sum`, амортизация — в `redemtion`.*
```python
flows = api.get_bond_cashflows("RU000A10DQA8", limit=10)
for f in flows:
    print(f"Дата: {f.get('date')}, Купон: {f.get('cupon_sum')}, Амортизация: {f.get('redemtion')}")
```

#### Информация об офертах (`get_bond_offers`)
Возвращает список оферт (put/call опционы на досрочный выкуп) для заданной бумаги (эндпоинт `get_offert`).
```python
offers = api.get_bond_offers("RU000A10DQA8")
```

#### Информация о дефолтах (`get_bond_defaults`)
Возвращает историю технических или реальных дефолтов по выпуску (эндпоинт `get_emission_default`).
```python
defaults = api.get_bond_defaults("RU000A10DQA8")
```

---

### 2. Рыночные котировки и сделки

#### Котировки Московской биржи (`get_market_trades`)
Запрос дневных свечей и истории торгов по облигации на Московской бирже (эндпоинт `get_tradings_new`).
```python
trades = api.get_market_trades(
    isin="RU000A10DQA8",
    date_from="2024-01-01",
    date_to="2024-05-01",
    limit=50
)
for t in trades:
    print(f"Дата: {t.get('date')}, Цена закрытия: {t.get('close')}%")
```

---

### 3. Индексы и макроэкономика
Все макроэкономические показатели запрашиваются через эндпоинт `get_index_value_new`. Класс содержит предопределенный словарь `AVAILABLE_INDICES` с ID популярных индексов.

#### Общий метод запроса индексов (`get_index_values`)
Позволяет запросить данные любого индекса по текстовому ключу (из словаря) или числовому ID.
```python
# Запрос по ключу
inflation = api.get_index_values("Inflation_Rate_Yearly", limit=12)

# Запрос по кастомному ID
custom_index = api.get_index_values(1599, limit=5) # 1599 - Индекс RGBI
```

#### Ключевая ставка ЦБ РФ (`get_key_rate`)
Возвращает историю изменения ключевой ставки Банка России.
```python
key_rate = api.get_key_rate(limit=5)
for kr in key_rate:
    print(f"Дата: {kr.get('date')}, Ставка: {kr.get('value')}%")
```

#### Ставка RUONIA (`get_ruonia_rate`)
Индикативная взвешенная ставка однодневных кредитов (депозитов) в рублях на межбанковском рынке.
```python
ruonia = api.get_ruonia_rate(limit=5)
```

#### Курсы валют Forex (`get_forex_rate`)
Запрос курсов валют к рублю. Доступные валютные пары: `USD_RUB`, `EUR_RUB`, `CNY_RUB`, `GBP_RUB`, `INR_RUB`, `JPY_RUB`.
```python
usd_rub = api.get_forex_rate("USD_RUB", limit=5)
```

#### Кривая доходности гособлигаций ОФЗ (`get_yield_curve`)
Получение значений доходности ОФЗ для фиксированных сроков погашения (например: `1W`, `1M`, `1Y`, `2Y`, `5Y`, `10Y`, `20Y` и т.д.).
```python
yield_5y = api.get_yield_curve("5Y", limit=5)
for y in yield_5y:
    print(f"Дата: {y.get('date')}, Доходность 5-летних ОФЗ: {y.get('value')}%")
```
